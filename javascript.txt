Java Script:

编程语言   面对对象的编程语言

对象：就是实实在在存在的一个实体，该实体在DOM中其实就是一个节点对象。拥有数据和属性以及方法。

属性：属性就是一个对象所拥有的数据（包括数据的类型和数据的值）

方法：就是对象所具有的功能。通过对象的功能使用对象的属性来实现一些操作。

脚本语言   解释性的语言  

解释性的
代码是不需要编译的，直接可以执行
代码按行执行，前面的代码不会受到后边的代码错误的影响

编译性的
先把代码编译为可执行文件 比如exe文件，然后才能执行。
假如语法有错误，或是代码在编译的时候出错了，就没有办法执行、

弱类型
变量本身没有类型，但是变量值有类型。即变量的类型是由变量的值决定的。

var a = 3;

强类型
变量在声明的时候必须执行变量的类型
integer a = 3;
